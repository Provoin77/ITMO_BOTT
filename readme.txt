Сервис доступен по ссылке - https://itmo-bot.onrender.com/api/request

Сервис, основан на использовании бесплатных ключей Google Search, OpenChat 3.5, Render (облако).
Поэтому к сожалению, при работе ключи и облако могут слетать, для их переактивации прошу написать в ТГ - https://t.me/Provoin
Соотвественно после переактивации, тестировать работу сервиса. Спасибо!

  "id": 2
}
